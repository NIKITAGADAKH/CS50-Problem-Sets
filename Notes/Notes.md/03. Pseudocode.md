# Pseudocode
* process of converting instructions into code is called pseudocode.
* Pseudocode is a human-readable version of your code.

### Pseudocode imp for following reasons 
* it allows you to think through the logic of your problem in advance. 
* you can later provide this information to others that are seeking to understand your coding decisions and how your code works.

### Features of Pseudocode
* some of these lines begin with verbs like pick up, open, look at called functions.
* some lines include statements like if or else if. called conditionals.
* there are expressions that can be stated as true or false, such as “person is earlier in the book.” We call these boolean expressions.
* statements like “go back to line 3.” We call these loops.
